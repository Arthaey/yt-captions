yt-captions
===========
Read Youtube transcript (captions) from any public Youtube video

####Requirements
PhantomJS http://phantomjs.org/download.html

####Usage
```bash
ruby yt_captions.rb http://www.youtube.com/watch?v=[video-id]
```
Where `[video-id]` is the Youtube video ID.
